### Подсказки по выученным на данный момент командам Git
Команды для консоли:<br>
**cd** выбор директории для перемещения<br>
**pwd** просмотр текущей директории<br>
**ls** просмотр содержимого текущей директории<br>
**git status** статус Git (проверка на предмет учтенный и сохраненных изменений)<br>
**git add** фиксация изменений в проекте<br>
**git commit -m 'коментарий'** коммит<br>
**git push** отправка текущего состояния после *git commit* в удаленный репозиторий<br>
**git clone** копирование удаленного репозитория на свой компьютер<br>
**git remote add origin** привязка локального репозитория к удаленному<br>
**git remote -v** проверка связанности репозиториев<br>
**git push -u origin main** первая загрузка на новый удаленный репозиторий<br>
